# 🎓 Rank2Seat – AP EAMCET Admission Probability Estimator

Rank2Seat is a **Machine Learning-based web application** that predicts the **probability of getting an engineering seat** during AP EAMCET counseling based on a student's rank and preferences.

The system learns from **multi-year counseling cutoff data** and provides **real-time admission probability predictions** to help students make better college selection decisions.

---

## 🚀 Features

- ✅ Admission probability prediction based on EAMCET rank
- ✅ Trained on **5 years of counseling data**
- ✅ Data preprocessing & feature engineering pipeline
- ✅ Real-time prediction using Machine Learning
- ✅ Interactive Streamlit web interface
- ✅ Decision support visualization (High / Moderate / Low Chance)

---

## 🧠 Machine Learning Workflow

1. Data Collection from multi-year counseling datasets  
2. Data Cleaning & Normalization  
3. Feature Engineering & Encoding  
4. Model Training using Scikit-learn  
5. Probability-based Prediction  
6. Real-time inference via Streamlit UI

---

## 🛠️ Tech Stack

- **Programming Language:** Python  
- **Libraries:** Pandas, NumPy, Scikit-learn  
- **Frontend/UI:** Streamlit  
- **Machine Learning:** Predictive Analytics Model  
- **Data Processing:** Feature Engineering & Normalization

---

## 📊 Dataset

- AP EAMCET Counseling Data (2019–2024)
- ~**1.8L+ processed training records**
- Includes:
  - College Name
  - Branch
  - Category
  - Gender
  - Closing Rank
